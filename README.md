# Weather in Cities 

A Next.js + TypeScript application that shows current weather for selected cities using the OpenWeatherMap API.

## Live Demo
https://weather-app-vercel-iota.vercel.app

## Features
- Add / remove cities
- Current weather in city cards
- Refresh weather for a single city
- Data persistence via LocalStorage
- Responsive (mobile-friendly)

## Tech Stack
- Next.js
- TypeScript
- React
- Redux Toolkit
- Axios
- SCSS
- Jest + Testing Library

## Getting Started

```bash
git clone https://github.com/asushko25/weather-app.git
cd weather-app
npm install
npm run dev
